📌 هيكل صفحة HTML

تعرّف على الهيكل الأساسي لأي صفحة ويب قبل أن تبدأ

الهيكل الكامل لصفحة HTML

كل صفحة HTML تبدأ بإعلان DOCTYPE ثم تحتوي على قسمين رئيسيين: <head> للمعلومات الخفية و<body> للمحتوى المرئي.

HTML
<!-- إعلان نوع الوثيقة --> <!DOCTYPE html> <html lang="ar"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>عنوان الصفحة</title> <link rel="stylesheet" href="style.css"> </head> <body> <!-- كل المحتوى المرئي يأتي هنا --> <h1>مرحباً بالعالم!</h1> </body> </html>
Meta Tags
وسوم Meta المهمة

تحدد معلومات الصفحة مثل الترميز، الوصف، والكلمات المفتاحية لمحركات البحث.

HTML
<meta charset="UTF-8"> <meta name="description" content="وصف الصفحة"> <meta name="keywords" content="HTML, تعلم"> <meta name="author" content="Ahmed Shawky">
Headings
العناوين h1 – h6

ست مستويات من العناوين. h1 للعنوان الرئيسي الأهم، وh6 للأصغر. يجب استخدام h1 مرة واحدة فقط في الصفحة.

HTML
<h1>عنوان رئيسي</h1> <h2>عنوان ثانوي</h2> <h3>عنوان فرعي</h3> <h4>عنوان أصغر</h4> <h5>عنوان خامس</h5> <h6>عنوان سادس</h6>
Comments
التعليقات

لا تظهر في الصفحة، تُستخدم لشرح الكود أو إيقاف تشغيله مؤقتاً.

HTML
<!-- هذا تعليق لا يظهر --> <!-- تعليق متعدد الأسطر --> <!-- <p>هذا الكود موقف</p> -->
Attributes
الخصائص (Attributes)

معلومات إضافية تُضاف داخل الوسم. كل خاصية لها اسم وقيمة.

HTML
<!-- id: معرف فريد --> <div id="main"></div> <!-- class: لمجموعة عناصر --> <p class="text-red">نص</p> <!-- style: تنسيق مباشر --> <p style="color:red">نص</p>
💡
تذكر: وسوم HTML تأتي دائماً في أزواج — وسم فتح <p> ووسم إغلاق </p> — إلا الوسوم الذاتية الإغلاق مثل <br> و<img> و<meta>

✍️ تنسيق النصوص

كل الوسوم المتعلقة بالنصوص والكتابة في HTML

Text Formatting
التنسيق الأساسي

وسوم لتغيير مظهر النصوص: عريض، مائل، خط تحت، خط منتصف.

HTML
<b>نص عريض (Bold)</b> <strong>مهم دلالياً</strong> <i>نص مائل (Italic)</i> <em>تأكيد دلالي</em> <u>خط تحتي</u> <s>نص محذوف</s> <mark>نص محدد</mark>
Sup & Sub
الأس والمؤشر

للكتابة فوق السطر (أسس رياضية) أو تحته (صيغ كيميائية).

HTML
<!-- أس رياضي: x² --> x<sup>2</sup> <!-- صيغة كيميائية: H₂O --> H<sub>2</sub>O <!-- ملاحظة قدم --> نص<sup>[1]</sup>
Quote
الاقتباسات

لعرض الاقتباسات من مصادر أخرى بشكل صحيح دلالياً.

HTML
<!-- اقتباس مضمّن --> <q>قول مأثور</q> <!-- اقتباس بلوك --> <blockquote cite="https://..."> نص الاقتباس الطويل </blockquote> <!-- مصدر --> <cite>اسم المرجع</cite>
Code & Pre
عرض الكود البرمجي

لعرض كود أو نص محافظاً على المسافات والتنسيق.

HTML
<!-- كود مضمّن --> <code>let x = 5;</code> <!-- كود بلوك محافظ التنسيق --> <pre> <code> function hello() { return "مرحبا"; } </code> </pre>
Links
الروابط بالتفصيل

وسم <a> للروابط مع جميع خياراته وطرق استخدامه.

HTML
<!-- رابط خارجي --> <a href="https://google.com" target="_blank" rel="noopener">Google</a> <!-- رابط بريد إلكتروني --> <a href="mailto:hi@test.com">راسلني</a> <!-- رابط هاتف --> <a href="tel:+201234567890">اتصل</a> <!-- رابط داخل الصفحة --> <a href="#section1">انتقل</a>
Span & Div
Span و Div

span لتنسيق جزء من النص، div لتقسيم الصفحة إلى كتل.

HTML
<!-- span: inline --> <p> اضغط <span style="color:red">هنا</span> </p> <!-- div: block --> <div class="box"> <p>محتوى القسم</p> </div>

🎬 الوسائط والصور

إضافة صور، فيديو، صوت، وخرائط iframe

Image
الصور بالتفصيل

إضافة صورة مع تحديد الحجم ونص بديل لإمكانية الوصول.

HTML
<img src="photo.jpg" alt="وصف الصورة" width="300" height="200" loading="lazy" > <!-- صورة من موقع آخر --> <img src="https://site.com/img.png" alt="صورة خارجية">
Video
الفيديو

تضمين فيديو مع أدوات التحكم وإمكانية تشغيل أكثر من صيغة.

HTML
<video width="640" controls autoplay muted loop poster="thumb.jpg" > <source src="video.mp4" type="video/mp4"> <source src="video.webm" type="video/webm"> </video>
Audio
الصوت

تضمين ملف صوتي مع دعم أكثر من صيغة.

HTML
<audio controls> <source src="song.mp3" type="audio/mpeg"> <source src="song.ogg" type="audio/ogg"> متصفحك لا يدعم الصوت </audio>
iFrame
تضمين iFrame

لتضمين صفحة أخرى أو خريطة Google أو فيديو يوتيوب.

HTML
<!-- يوتيوب --> <iframe src="https://www.youtube.com/embed/ID" width="560" height="315" allowfullscreen ></iframe> <!-- خريطة جوجل --> <iframe src="https://maps.google.com/maps?..." ></iframe>
Figure
Figure و Figcaption

لعرض الصور مع تسميات توضيحية بشكل دلالي صحيح.

HTML
<figure> <img src="sunset.jpg" alt="غروب الشمس"> <figcaption> غروب الشمس على البحر </figcaption> </figure>
Picture
وسم Picture المتجاوب

لتقديم صور مختلفة لأحجام شاشات مختلفة (Responsive Images).

HTML
<picture> <source media="(max-width: 600px)" srcset="small.jpg"> <source media="(max-width: 1200px)" srcset="medium.jpg"> <img src="large.jpg" alt="صورة"> </picture>

📋 القوائم والجداول

تنظيم البيانات في قوائم وجداول منسقة

Unordered List
القائمة غير المرتبة

قائمة بنقاط بدون ترقيم، تستخدم للعناصر التي لا يهم ترتيبها.

HTML
<ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ul>
Ordered List
القائمة المرتبة

قائمة مرقمة، تستخدم للخطوات المتسلسلة أو الترتيب المهم.

HTML
<ol type="1" start="1"> <li>الخطوة الأولى</li> <li>الخطوة الثانية</li> <li>الخطوة الثالثة</li> </ol> <!-- type="A" للحروف، "i" للروماني -->
Nested List
القوائم المتداخلة

قوائم داخل قوائم لتمثيل التسلسل الهرمي.

HTML
<ul> <li>Frontend <ul> <li>HTML</li> <li>CSS</li> </ul> </li> <li>Backend</li> </ul>
Description List
قائمة التعريفات

لعرض المصطلحات وتعريفاتها مثل القاموس.

HTML
<dl> <dt>HTML</dt> <dd>لغة ترميز النص التشعبي</dd> <dt>CSS</dt> <dd>أوراق أنماط الصفحات</dd> </dl>
الجداول بالكامل

الجدول الكامل مع رأس، جسم، تذييل، ودمج الخلايا.

HTML
<table border="1"> <caption>جدول الدرجات</caption> <thead> <tr> <th>الاسم</th> <th>الدرجة</th> <th>التقدير</th> </tr> </thead> <tbody> <tr> <td>أحمد</td> <td colspan="2">95 - ممتاز</td> </tr> <tr> <td rowspan="2">محمد</td> <td>80</td> <td>جيد جداً</td> </tr> </tbody> <tfoot> <tr> <td colspan="3">المجموع</td> </tr> </tfoot> </table>

📝 النماذج والمدخلات

جمع البيانات من المستخدمين

نموذج شامل
HTML
<form action="/submit" method="POST"> <!-- حقل نص --> <label for="name">الاسم:</label> <input type="text" id="name" name="name" placeholder="أدخل اسمك" required> <!-- بريد إلكتروني --> <input type="email" placeholder="البريد"> <!-- كلمة مرور --> <input type="password" minlength="8"> <!-- رقم --> <input type="number" min="0" max="100"> <!-- تاريخ --> <input type="date"> <!-- لون --> <input type="color"> <!-- نطاق (slider) --> <input type="range" min="0" max="100"> <!-- مربع اختيار --> <input type="checkbox"> أوافق <!-- زر راديو --> <input type="radio" name="gender"> ذكر <input type="radio" name="gender"> أنثى <!-- قائمة منسدلة --> <select name="city"> <option value="cairo">القاهرة</option> <option value="alex">الإسكندرية</option> </select> <!-- منطقة نص --> <textarea rows="4" cols="50"></textarea> <!-- رفع ملف --> <input type="file" accept=".jpg,.pdf"> <!-- زر إرسال --> <button type="submit">إرسال</button> </form>
💡
استخدم دائماً <label> مع كل <input> وربطهما عبر خاصية for و id — هذا يحسن إمكانية الوصول ويجعل النقر على التسمية يُفعّل الحقل.

🏗️ Semantic HTML5

الوسوم الدلالية التي تجعل الكود أوضح لمحركات البحث وقارئات الشاشة

هيكل صفحة Semantic كاملة
HTML
<body> <header> <nav> <a href="/">الرئيسية</a> </nav> </header> <main> <section> <h2>قسم رئيسي</h2> <article> <h3>مقال</h3> <p>محتوى المقال...</p> </article> </section> <aside> <!-- شريط جانبي --> </aside> </main> <footer> <p>&copy; 2025 Ahmed Shawky</p> </footer> </body>
Details & Summary
الأكورديون الأصلي

لإنشاء أقسام قابلة للطي والتوسيع بدون JavaScript.

HTML
<details> <summary>ما هو HTML؟</summary> <p> HTML هي لغة ترميز النص التشعبي لبناء صفحات الويب. </p> </details>
Dialog
نافذة Dialog

نافذة منبثقة أصلية في HTML5 بدون مكتبات.

HTML
<dialog id="myDialog"> <p>مرحباً بك!</p> <button onclick="myDialog.close()"> إغلاق </button> </dialog> <button onclick="myDialog.showModal()"> افتح النافذة </button>
Progress & Meter
شريط التقدم

لعرض تقدم عملية أو قيمة ضمن نطاق معين.

HTML
<!-- شريط تحميل --> <progress value="70" max="100"> </progress> <!-- مقياس --> <meter value="6" min="0" max="10" low="3" high="8"> </meter>

⚡ مواضيع متقدمة

Canvas، SVG، Accessibility، SEO، وأكثر

Canvas
رسم بـ Canvas

رسم الأشكال والصور ديناميكياً باستخدام JavaScript.

HTML + JS
<canvas id="myCanvas" width="400" height="200"> </canvas> <script> const canvas = document.getElementById('myCanvas'); const ctx = canvas.getContext('2d'); ctx.fillStyle = '#8a2be2'; ctx.fillRect(10, 10, 150, 80); </script>
SVG
رسومات SVG

رسومات متجهية قابلة للتكبير داخل HTML مباشرة.

SVG
<svg width="200" height="100"> <!-- دائرة --> <circle cx="50" cy="50" r="40" fill="purple"/> <!-- مستطيل --> <rect x="100" y="10" width="80" height="80" fill="#b266ff"/> <!-- نص --> <text x="10" y="95" fill="white">SVG</text> </svg>
Accessibility
إمكانية الوصول (a11y)

جعل موقعك قابلاً للاستخدام لذوي الإعاقة باستخدام ARIA.

HTML
<!-- دور ARIA --> <div role="navigation" aria-label="القائمة الرئيسية"> </div> <!-- زر مع وصف --> <button aria-label="إغلاق النافذة"> ✕ </button> <!-- وصف مخفي بصرياً --> <span class="sr-only"> مطلوب </span>
Data Attributes
خصائص data-*

لتخزين بيانات مخصصة داخل عناصر HTML لاستخدامها في JavaScript.

HTML + JS
<div data-id="42" data-name="أحمد" data-role="admin" >المستخدم</div> <script> const el = document.querySelector('div'); console.log(el.dataset.id); // 42 console.log(el.dataset.name); // أحمد </script>
Template
وسم Template

لتعريف قوالب HTML تُستخدم بواسطة JavaScript دون عرضها مباشرة.

HTML
<template id="card-tpl"> <div class="card"> <h3 class="title"></h3> </div> </template> <script> const tpl = document.getElementById('card-tpl'); const clone = tpl.content.cloneNode(true); document.body.appendChild(clone); </script>
Open Graph
Open Graph (مشاركة سوشيال)

تحديد كيفية ظهور صفحتك عند مشاركتها على وسائل التواصل الاجتماعي.

HTML
<head> <meta property="og:title" content="عنوان الصفحة"> <meta property="og:description" content="وصف قصير"> <meta property="og:image" content="https://site.com/img.jpg"> <meta property="og:url" content="https://site.com"> </head>

3️⃣ أي وسم يُستخدم لإنشاء قائمة مرتبة (مرقمة)؟
4️⃣ ما الفرق بين <b> و <strong>؟
5️⃣ أين يجب وضع وسوم <meta> و <title>؟
